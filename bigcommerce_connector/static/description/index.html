Is this connector compatible with any Odoo version?

At this time, our Odoo connector is compatible with Odoo 13, Odoo 14, and Odoo 15 only. If you have an older version of Odoo, please contact Novobi's customer support for additional information. 

What is the process for DIY installation?

1. Download the modules and install them to the Odoo database.

2. Set up an API account on the BigCommerce store and configure the credentials to Odoo.

3. Set up product mapping between Odoo and BigCommerce.

4. Set up workflow and automation for order sync.

5. Configure for inventory sync.

All of these should be done and verified on a staging database before installing on the live database.
